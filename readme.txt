Системные требования:
    PHP: Версия 7.4 (или совместимая с проектом).
    База данных: PostgreSQL 13.
    Веб-сервер: Open Server Panel версии 5.4.3 (или аналогичный).

Импортируйте дамп базы данных.

Скопируйте файлы проекта в папку domains Open Server Panel (например, C:/OpenServer/domains/localhost/project).

Запуск сервера:

Откройте Open Server Panel и запустите сервер.
Перейдите в браузере по адресу http://localhost/project/visualization.php (для визуализации первой части задания)
Перейдите в браузере по адресу http://localhost/project/admin_visualization.php (для визуализации второй части задания)
